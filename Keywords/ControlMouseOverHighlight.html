ControlMouseOverHighlight

Format: ControlMouseOverHighlight FrameName,

Example: ControlMouseOverHighlight "BorderedButtonMouseOverHighlightTemplate",

Purpose: Define the HIGHLIGHT-Frame beeing displayed while the parentFrame is hovered with the Mouse. A Button should have ControlStyle "HIGHLIGHTONMOUSEOVER" to use this.

Used in: BUTTON, GLUEBUTTON, TEXTBUTTON, GLUETEXTBUTTON
